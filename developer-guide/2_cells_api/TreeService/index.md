This service is the main entry point for listing files and folders, getting stats about them, creating folders, etc. Please note that it does not provide access to the actual content (upload/download), which must be accessed using the S3 API.

- [GET /a/tree/stat/{Node}](/cells-v4/developer-guide/rest-api/tree-service/get-a-tree-stat-node/)
- [POST /a/tree/create](/cells-v4/developer-guide/rest-api/tree-service/post-a-tree-create/)
- [POST /a/tree/delete](/cells-v4/developer-guide/rest-api/tree-service/post-a-tree-delete/)
- [POST /a/tree/restore](/cells-v4/developer-guide/rest-api/tree-service/post-a-tree-restore/)
- [POST /a/tree/selection](/cells-v4/developer-guide/rest-api/tree-service/post-a-tree-selection/)
- [POST /a/tree/stats](/cells-v4/developer-guide/rest-api/tree-service/post-a-tree-stats/)
